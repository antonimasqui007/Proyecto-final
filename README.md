# Generador Seguro de Contraseñas

## Nombre del proyecto
**El impacto de las nuevas tecnologías en la sociedad: desarrollo y proyección de soluciones informáticas**

---

## Integrantes: Antoni Masqui
---

## Objetivo del sistema

Desarrollar un programa en Python que genere contraseñas seguras y personalizadas, promoviendo el uso responsable de la tecnología para proteger la identidad digital de las personas en la sociedad actual.

---

## Descripción de funcionalidades

| Funcionalidad | Descripción |
|---|---|
| Menú interactivo | El usuario configura la contraseña paso a paso desde la consola |
| Elección de longitud | Define cuántos caracteres tendrá la contraseña (8 a 64) |
| Tipos de caracteres | Elige entre mayúsculas, minúsculas, números y símbolos |
| Generación múltiple | Puede generar hasta 10 contraseñas en una sola ejecución |
| Evaluador de fortaleza | Califica la contraseña como Débil, Media, Fuerte o Muy fuerte |
| Historial de sesión | Muestra un resumen de todas las contraseñas generadas al finalizar |
